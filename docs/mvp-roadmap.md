# MVP Roadmap

## Phase 1

- user onboarding
- admin login
- manual match creation
- vote casting and vote updates before deadline
- result declaration
- settlement generation

## Phase 2

- fixture sync
- auto poll open and lock jobs
- final team list display
- carry-forward history

## Phase 3

- rivalry title generator
- richer IPL visuals
- team-themed onboarding and poll backgrounds
- admin analytics

## Phase 4

- optional WhatsApp notification bridge
- invite-only onboarding
- audit logs
- season leaderboard
